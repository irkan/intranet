Установка Intranet EXXO.ru на компьютер, работающий под Windows

Для запуска сервера, дважды кликните по скрипту start.bat

ВНИМАНИЕ: Есле на компьютере есть работающие службы Tomcat или Postgresql,
то их необходимо остановить перед запуском сервера.


Откройте ваш браузер. В адресной строке наберите http://localhost/


Чтобы зайти с другого компьютера сети, нужно набрать:
http://Адрес компьютера с программой/


В базу данных загружен Демо-портал, позволяющий ускорить освоение программы.
